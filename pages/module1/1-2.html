↑ WPS演示界面区域标注示意图（标号对应下方说明）
①
标题栏
显示当前文件名称和软件名称。 双击标题栏可以最大化/还原窗口； 拖动标题栏可以移动窗口位置。
②
选项卡 / 功能区
顶部的"开始、插入、设计、动画、幻灯片放映、视图"等标签。 点击不同选项卡会切换下方显示的工具按钮组， 所有操作命令都在这里找到。
③
工具按钮区（功能区主体）
根据当前选项卡显示对应的操作按钮，如字体、段落、插入图片等。 鼠标悬停在按钮上会显示功能提示（Tooltip），方便快速了解每个按钮的用途。
④
幻灯片缩略图区
左侧面板，以小图形式显示所有幻灯片。 点击缩略图可快速跳转到对应幻灯片； 右键缩略图可以插入、删除、复制、移动幻灯片； 直接拖拽缩略图可以调整幻灯片顺序。
⑤
幻灯片编辑区（主编辑区）
中央最大的区域，用于编辑当前幻灯片的内容。 在这里可以输入文字、插入图片、调整布局； 点击元素可以选中并编辑，拖拽可以移动位置。
⑥
任务窗格
右侧可折叠的面板，根据当前操作显示相关设置选项， 如动画属性、图片格式、主题设计等。 不需要时可以关闭，给编辑区留出更多空间。
⑦
状态栏
底部信息栏，显示当前幻灯片页码、总页数、视图模式切换按钮和缩放比例。 点击右下角的"+""-"可以快速调整编辑区的缩放大小。
💡
小技巧： 如果界面看起来太小，可以按住 Ctrl 并滚动鼠标滚轮来缩放编辑区； 也可以拖动状态栏右下角的缩放滑块快速调整显示比例。
⌨️ 必须掌握的常用快捷键

熟练使用快捷键可以大幅提升操作效率。 下面这些快捷键是WPS演示中最常用的，建议记住！

📁 文件操作
Ctrl + N
新建演示文稿
Ctrl + O
打开文件
Ctrl + S
保存文件
Ctrl + W
关闭文件
Ctrl + P
打印文件
F12
另存为
✏️ 编辑操作
Ctrl + Z
撤销上一步操作
Ctrl + Y
重做（恢复撤销）
Ctrl + C
复制所选内容
Ctrl + V
粘贴内容
Ctrl + X
剪切内容
Ctrl + A
全选当前幻灯片元素
Ctrl + D
复制并粘贴（直接复制）
Delete
删除选中元素
🖼️ 幻灯片操作
Enter
在当前幻灯片后新建一张
F5
从第一张开始放映
Shift + F5
从当前幻灯片开始放映
Esc
退出放映模式
Ctrl + M
插入新幻灯片
Page↓
跳转到下一张幻灯片
🏆
记忆小窍门： 大多数快捷键都是英文单词首字母！ N = New（新建），S = Save（保存），C = Copy（复制）， V = paste（粘贴），Z = undo（撤销）。 理解含义比死记硬背更有效！
👁️ 四种视图模式

WPS演示提供了多种视图模式，不同模式适合不同的操作场景。 通过底部状态栏或"视图"选项卡可以切换。

📝
普通视图
默认视图，左侧显示缩略图， 中间是编辑区，下方可写备注。 日常制作PPT就在这个视图下完成。
最常用 ⭐
🗂️
幻灯片浏览视图
以缩略图方式显示所有幻灯片， 方便整体查看排列效果、 拖拽调整顺序、批量操作幻灯片。
整理顺序用
📒
备注页视图
幻灯片下方显示演讲备注区域， 可以为每张幻灯片添加演讲要点， 放映时演讲者可以看到但观众看不见。
演讲备注用
🖥️
阅读视图
在窗口内全屏预览演示效果， 不占满整个显示器，方便一边查看效果 一边修改内容，按 Esc 退出。
预览效果用
⭐
切换视图的方法： 点击底部状态栏右侧的视图图标， 或点击顶部"视图"选项卡进行切换。 普通视图 是我们日常编辑的主要视图， 记住它就够啦！
📋 本节知识小结
① WPS演示界面分为：标题栏、功能区、工具按钮区、缩略图区、 编辑区、任务窗格、状态栏共 7个区域
② 最重要的快捷键： Ctrl+S 保存、 Ctrl+Z 撤销、 Ctrl+C/V 复制粘贴、 F5 放映
③ 四种视图模式：普通视图（编辑）、 幻灯片浏览（整理）、 备注页（备注）、 阅读视图（预览）
